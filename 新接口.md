这个接口会返回玩家的比赛历史，“items”数组下的每个子元素就是一场比赛记录。
下表为PlayerMatch需要的数据：

| PlayerMatch | JSON返回内容                                                                                             |
| ----------- | ---------------------------------------------------------------------------------------------------- |
| nickName    | 通过当前的枚举PlayerEnum的uuid找到items.teams里面的队伍，一般都是有两个队伍，队伍字段里有players数组找到与当前玩家一样的player_id，获取他的nickname值。 |
| team        | 在找nickName的时候你知道了当前玩家所在teams对象里的哪个队伍，获取到队伍的名字如“faction1”。                                            |
| matchScore  | results.score字段里有各队伍的比分，保存的格式为：玩家所在队伍的比分/对手比分                                                        |
| matchResult | 比赛结果，results.score字段里有“winner”字段值是胜利的队伍，如果胜利队伍是当前玩家所属的队伍就保存为WIN否则为LOSS                               |
| matchId     | "match_id"字段                                                                                         |
| roomUrl     | 拼接字符串"https://www.faceit.com/en/cs2/room/" + matchId+"/scoreboard"                                   |
| timestamp   | “started_at”字段                                                                                       |
